## ITAI 1371 Intro to ML Repository
This repository contains coursework and projects for the Introduction to AI/Machine Learning course (ITAI 1371) at Houston City College.

# Course Repo/
├── Final_project/               # Folder containing Final project
├── midterm_project/             # Folder containing MidTerm project
├── modules/                     # Folder containing Lab Modules
└── README.md                    # This file

# Modules
Module 03: ML Workflows and Types of Learning
Module 04: Working with Data and EDA
Module 05: Data Preparation and Feature Engineering
Module 06 - Regression and Classification
Module 07 - Evaluation Machine Learning Models
Module 08 - Overfitting, Underfitting, and Regularization
Module 09 - Ensemble Methods
Module 10 - Unsupervised Learning
Module 11 - Hyperparameter tuning and AutoML
Module 12 - Ethics, Fairness, Bias
Module 13 - ML Pipelines
Module 14 - Real-World Applications

# Usage
Each module folder contains its own notebooks and supporting files with specific instructions and exercises.

# Course Information
This repository is part of an Introduction to AI/Machine Learning course, ITAI 1371 at Houston City College. The materials cover fundamental concepts and practical applications of machine learning algorithms.
